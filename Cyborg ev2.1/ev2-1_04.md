# 📘 **Cyborg ev2.1 Patch Language Grammar**  
*(Formal Syntax Specification)*





---

## **Overview**

This page defines the **complete grammar** for Cyborg ev2.1 patch files.  
The grammar is expressed using **EBNF‑style notation**, consistent with MSDN documentation for:

- PE/COFF  
- MZ executable format  
- MASM syntax  
- Win32 resource scripts  

This grammar is used by:

- Constellation validators  
- Cyborg patch parsers  
- LLM‑HEX interpreters  
- Robot ingestion engines  

---

## **1. File Structure**

```
patch-file   ::= header-block
                 { asm-block | c-block | llm-block | bind-block }
```

---

## **2. Header Block**

```
header-block ::= "MZ-PATCH" NEWLINE
                  "TARGET:" path NEWLINE
                  "MODE:"   mode NEWLINE
                  "ARCH:"   arch NEWLINE
                  "VERSION:" version NEWLINE

mode         ::= "READ_ONLY" | "SIMULATE" | "APPLY"
arch         ::= "x86" | "x86_64"
version      ::= "EV2.1"
path         ::= <any valid file path>
```

---

## **3. ASM Block**

```
asm-block    ::= "[ASM" [ ":" label ] "]" NEWLINE
                 { asm-statement }
                 "ENDASM"

asm-statement ::= org-stmt
                | hook-block
                | instruction

org-stmt      ::= "ORG" address

hook-block    ::= "HOOK" "AT" address NEWLINE
                  { instruction }
                  "ENDHOOK"

instruction   ::= mnemonic operand-list

mnemonic      ::= "MOV" | "PUSH" | "POP" | "CALL" | "JMP"
                | "CMP" | "TEST" | "RET" | "NOP"

operand-list  ::= operand { "," operand }
operand       ::= register | immediate | address

register      ::= "AX" | "BX" | "CX" | "DX"
                | "SP" | "BP" | "SI" | "DI"

address       ::= hex-value
immediate     ::= hex-value
label         ::= identifier
```

---

## **4. C Block**

```
c-block       ::= "[C]" NEWLINE
                  { c-function }

c-function    ::= abi return-type identifier "(" parameter-list ")" "{" 
                  c-body 
                  "}"

abi           ::= "__stdcall" | "__fastcall" | "__thiscall" | "__cycall"

return-type   ::= identifier
parameter-list ::= [ parameter { "," parameter } ]
parameter     ::= type identifier
type          ::= identifier

c-body        ::= <freeform text until matching brace>
```

---

## **5. LLM‑HEX Block**

```
llm-block     ::= "[LLM-HEX]" NEWLINE
                  "FORMAT:" format NEWLINE
                  "MODEL:"  model NEWLINE
                  "MODE:"   llm-mode NEWLINE
                  [ "SIZE:" hex-value NEWLINE ]
                  [ data-block ]

format        ::= "GGUF" | "LLAMA"
model         ::= identifier | path
llm-mode      ::= "EMBED" | "REFERENCE"

data-block    ::= "DATA:" NEWLINE
                  hex-stream
                  "ENDDATA"

hex-stream    ::= { hex-byte }
hex-byte      ::= 2 hex-digits
```

---

## **6. BIND Block**

```
bind-block    ::= "[BIND]" NEWLINE
                  { bind-entry }

bind-entry    ::= "ASM:" label NEWLINE
                  "C:"   identifier NEWLINE
                  [ "LLM:" model NEWLINE ]
                  [ "MODE:" bind-mode NEWLINE ]

bind-mode     ::= "STRICT" | "ADAPTIVE"
```

---

## **7. Identifiers & Values**

```
identifier    ::= letter { letter | digit | "_" }
hex-value     ::= "0x" hex-digits
hex-digits    ::= hex-digit { hex-digit }
hex-digit     ::= "0".."9" | "A".."F" | "a".."f"
letter        ::= "A".."Z" | "a".."z"
digit         ::= "0".."9"
```

---

## **Remarks**

- Grammar is case‑sensitive.  
- Blocks may appear in any order after the header.  
- Unknown blocks must be ignored by forward‑compatible parsers.  
- All offsets are symbolic until resolved by Constellation.  
- LLM‑HEX blocks must be validated before ingestion.  

---

## **Example (Valid Patch File)**

```
MZ-PATCH
TARGET: /Cyborg/engine/cyborg.exe
MODE:   APPLY
ARCH:   x86_64
VERSION: EV2.1

[ASM:HOOK_MAIN]
HOOK AT 0x401050
    PUSH EAX
    CALL 0x500000
    POP  EAX
ENDHOOK

[C]
__stdcall void cyborg_main_hook(void) {
    log_event("Cyborg ev2.1 patch engaged");
}

[LLM-HEX]
FORMAT: LLAMA
MODEL:  cyborg://models/patch/ev2.1/assistant
MODE:   REFERENCE

[BIND]
ASM:   HOOK_MAIN
C:     cyborg_main_hook
LLM:   cyborg://models/patch/ev2.1/assistant
MODE:  ADAPTIVE
```
