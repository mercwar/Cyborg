# eV2 COMPILER SPEC

The eV2 compiler transforms EV2 syntax into Fire‑Lang compatible IR.

## COMPILER STAGES
- LEXER  
- PARSER  
- SEMANTIC_PASS  
- ADDRESS_RESOLUTION  
- IR_EMIT  
- FIRELINK_BIND  

## IR FORMAT
IR.OPCODE  
IR.PARAM  
IR.ADDR  

## EXAMPLE IR EMISSION
OP_EXE  
ADDR 0x0300
