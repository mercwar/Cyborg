# eV2 INTERRUPT MAP

Interrupts unify EV1 event routing with Fire‑Lang execution.

## INTERRUPT VECTOR TABLE
INT 0x01 → KEY_EVENT  
INT 0x02 → MOUSE_EVENT  
INT 0x03 → WINDOW_RESIZE  
INT 0x10 → FIRE_CORE_WAKE  
INT 0xFF → FINALIZER_TRIGGER

## HANDLER SYNTAX
INT.HANDLE 0x10 -> fire.exec
