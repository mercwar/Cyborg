# eV2 ERROR CODES

Standardized EV2 error codes for runtime and compiler.

ERR_01 — INVALID_HEX  
ERR_02 — SEED_WEIGHT_INVALID  
ERR_03 — CDATA_LENGTH_MISMATCH  
ERR_04 — ADDR_LOCK_FAIL  
ERR_05 — FIRE_CORE_UNAVAILABLE  
ERR_FF — FINALIZER_TRIGGERED
