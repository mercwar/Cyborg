# STARFIRE Bridge: CBORD → CYBORD → CBORG → CYBORG → STARFIRE

## Overview

This patch set wires the lower continuum layers (CBORD, CYBORD, CBORG, CYBORG) into the STARFIRE astral layer using a small, consistent bridge surface.

## Patched Files

- `cbord.core.js`  
- `cybord.core.js`  
- `cborg.core.js`  
- `cyborg.core.js`  

Each now exposes:

- `toContinuum(type, payload)` → returns a canonical continuum event:
  - `source_layer`
  - `event_type`
  - `payload`
  - `timestamp`

## Bridge Logic

- `starfire.bridge.js`:
  - `STARFIRE_BRIDGE.fromLayer(layer, type, payload)`:
    - Calls the appropriate `toContinuum` on CBORD/CYBORD/CBORG/CYBORG.
    - Passes the resulting event into `STARFIRE.ingestContinuumEvent(...)`.

## JSON Mapping

- `continuum.bridge.json`:
  - Declares how continuum events map to STARFIRE stars and routes:
    - `CYBORG.merged` → `POLARIS → YILDUN`
    - `CBORG.dispatch` → `YILDUN → KOCHAB`
    - `INCINTUUM.gateway_output` → `KOCHAB → PHERKAD`
    - `CBORD.runtime` → `PHERKAD → POLARIS`

## Usage Example

```js
// From CYBORG:
const merged = CYBORG.fire("merged", { foo: "bar" });
const astral = STARFIRE_BRIDGE.fromLayer("CYBORG", "merged", merged);

// Direct STARFIRE orchestration:
const result = STARFIRE.orchestrate("merged", { starfire: true });
console.log(result);
