      <h2>CBORD IR (Mid‑Level C‑Like IR)</h2>
      <p>CBORD IR is the mid‑level representation used before emitting C, ASM, or higher‑level targets.</p>

      <h3>IR Structure</h3>
      <pre><code>{
  "CBORD_IR": {
    "KIND": "MID-LEVEL_C-LIKE_IR",
    "NODES": [
      "MODULE", "FUNCTION", "BLOCK", "PHI",
      "DECL", "ASSIGN", "CALL", "RETURN",
      "IF", "WHILE", "FOR", "BREAK", "CONTINUE",
      "BIN_OP", "UN_OP", "LOAD", "STORE",
      "CONST", "VAR", "ADDR", "DEREF",
      "ASM_INLINE", "EMIT_C"
    ]
  }
}</code></pre>

      <h3>Lowering Rules (CBORD → C / ASM)</h3>
      <pre><code>/* FILE: CBORD_LOWERING_RULES.irf */

CBORD_LOWERING_RULES
{
    LOWER BIN_OP {
        +  → C: (left + right), ASM: add
        -  → C: (left - right), ASM: sub
        *  → C: (left * right), ASM: imul
        /  → C: (left / right), ASM: idiv
        == → C: (left == right), ASM: cmp + je
        != → C: (left != right), ASM: cmp + jne
    }

    LOWER IF {
        C: if(cond) { then } else { else }
        ASM:
            cmp cond
            je THEN_LABEL
            jmp ELSE_LABEL
    }

    LOWER WHILE {
        C: while(cond) { body }
        ASM:
            LOOP_START:
                cmp cond
                jne LOOP_END
                body
                jmp LOOP_START
            LOOP_END:
    }

    LOWER FUNCTION {
        C: standard C function
        ASM:
            push rbp
            mov rbp, rsp
            ...body...
            pop rbp
            ret
    }

    LOWER ASM_INLINE {
        C: __asm__(text)
        ASM: text
    }

    LOWER EMIT_C {
        C: c_text
        ASM: (none)
    }
}</code></pre>
    </section>

    <section>
      <h2>CYBORD Layer (Executable CBORD Programs)</h2>
      <p>CYBORD is the executable tier built on CBORD IR, capable of emitting C, C++, Java, and ASM.</p>

      <h3>CYBORD Source Example</h3>
      <pre><code>/* FILE: example.cybord */

CBORD BEGIN

int add(int a, int b) {
    return a + b;
}

CBORD EMIT_CPP {
    // C++ wrapper
    #include &lt;iostream&gt;
    int main() {
        std::cout &lt;&lt; add(1, 2) &lt;&lt; std::endl;
        return 0;
    }
}

CBORD EMIT_JAVA {
    // Java wrapper
    public class Main {
        public static void main(String[] args) {
            System.out.println(add(1, 2));
        }
    }
}

CBORD END</code></pre>

      <h3>CYBORD → C Emission</h3>
      <pre><code>/* Emitted C from CYBORD */

int add(int a, int b) {

}</code></pre>
