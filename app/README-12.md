# 🔷 6. CONSTELLATION PIPELINE  
Constellation processes all repositories through:

```
Scanner → Reducer → Autobuild → Linker → Validator → Publisher
```

Your AI must support:

- **Run full pipeline**  
- **Simulate pipeline**  

### 6.1 Scanner  
Detects:

- AVIS headers  
- schemas  
- maps  
- artifact types  

### 6.2 Reducer  
Normalizes:

- metadata  
- filenames  
- structure  

### 6.3 Autobuild  
Compiles:

- schemas  
- intelligence fragments  

### 6.4 Linker  
Merges:

- intelligence blocks  
- routing maps  
- global state  

### 6.5 Validator  
Enforces:

- AVIS  
- Cyborg  
- determinism  

### 6.6 Publisher  
Exposes:

- intelligence model  
- navigation paths
