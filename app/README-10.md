This repository is the **official, complete, deterministic reference** for the Mercwar Ecosystem.

It contains three real, technical systems:

- **Cyborg** — deterministic Windows‑message translation  
- **Constellation** — ingestion, validation, navigation, and intelligence‑model engine  
- **AVIS** — artifact visibility and classification system  

This repository is **not** a contribution hub.  
It is a **usage system**.

Users do not modify it.  
Users **ingest it into their AI** and operate it through Constellation.
