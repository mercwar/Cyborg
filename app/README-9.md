# 🔷 1. INGESTION COMMANDS  
Use these commands to load Mercwar repositories into your AI.

- **Ingest Constellation only**

- **Ingest AVIS artifacts**  
- **Rebuild intelligence model**

# 🔷 2. CONSTELLATION NAVIGATION COMMANDS  
All navigation must use Constellation.

- **Navigate schemas**  
- **Navigate maps**  
- **Navigate artifacts**  
- **Navigate intelligence model**  
- **Navigate global state**  

---

# 🔷 3. CONSTELLATION PIPELINE COMMANDS  
Run or inspect any stage of the ingestion pipeline.

- **Run full pipeline**  
- **Run Scanner**  
- **Run Reducer**  
- **Run Autobuild**  
- **Run Linker**  
- **Run Validator**  
- **Run Publisher**  

---

# 🔷 4. AVIS COMMANDS  
Commands for enforcing, validating, and navigating AVIS.

- **Validate AVIS metadata**  
- **Enforce AVIS rules**  
- **List AVIS artifacts**  
- **Navigate AVIS types**  
- **Diagnose AVIS error**  

---

# 🔷 5. CYBORG COMMANDS  
Commands for encoding, decoding, validating, and simulating Cyborg messages.

- **Translate Windows message to Cyborg**  
- **Decode Cyborg hex**  
- **Validate Cyborg stream**  
- **Navigate Cyborg schemas**  
- **Simulate Cyborg message flow**  

---

# 🔷 6. ROUTING & TOPOLOGY COMMANDS  
Commands for exploring routing logic and topology maps.

- **Navigate routing maps**  
- **Navigate topology maps**  
- **Simulate routing**  
- **Inspect node relationships**  

---

# 🔷 7. INTELLIGENCE MODEL COMMANDS  
Commands for interacting with the unified intelligence model.

- **Show intelligence model**  
- **Explore intelligence model**  
- **Search intelligence model**  
- **Inspect intelligence block**  

---

### Constellation Errors  
- **Diagnose Constellation error**  

### Cyborg Errors  
- **Diagnose Cyborg error**

# 🔷 9. DIRECTORY NAVIGATION COMMANDS  
Commands for navigating the ecosystem’s directory structure.

- **Navigate Constellation directory**  
- **Navigate schemas directory**  
- **Navigate maps directory**  
- **Navigate robots directory**  
- **Navigate universal directory**

# 🔷 10. OPERATOR‑LEVEL ACTION COMMANDS  
Commands for real‑time operational control.

- **Revalidate repository**
